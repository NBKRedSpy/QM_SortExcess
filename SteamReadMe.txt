[h1]Quasimorph SortExcess[/h1]


Scrolling through multiple pages of the same item just to find the 3rd item in the list?

This mod changes the sort so that each item type will have a maximum of eight slots filled at the top of inventory screen, and then the rest will be put at the bottom.

The number of slots can be configured.  See the Configuration section.

[h1]Requirements[/h1]

Requires the Improved Item Sort mod.
On Steam: https://steamcommunity.com/sharedfiles/filedetails/?id=3287329716&searchtext=sort
On GitHub: https://github.com/NBKRedSpy/QM_ImprovedSort

[h1]Configuration[/h1]

The configuration file will be created on the first game run and can be found at [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\SortExcess\config.json[/i].
[table]
[tr]
[td]Name
[/td]
[td]Default
[/td]
[td]Description
[/td]
[/tr]
[tr]
[td]MaxItems
[/td]
[td]8
[/td]
[td]The maximum number of items to put to the top of the sort.
[/td]
[/tr]
[/table]
